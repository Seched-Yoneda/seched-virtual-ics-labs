# Network Terminology Standards

ドキュメント作成や翻訳において、ネットワーク関連の用語を扱う場合は、以下のルールを厳密に守ってください。

## 1. 基本ルール
- **ネットワークアダプタ (Network Adapter)**: 常に「**アダプタ**」と表記する。
- **ネットワークインタフェース (Network Interface)**: 常に「**インターフェース**」と表記する。

## 2. コンテキスト（文脈）による使い分け
- **ハードウェア寄り（物理的な構成や仮想マシンの設定）の場合**: 「**アダプタ**」とする。
  - *例*: VirtualBoxのネットワーク設定、物理NIC、USBネットワークアダプタなど
- **OS寄り（論理的な構成やデバイス名）の場合**: 「**インターフェース**」とする。
  - *例*: Linuxの `eth0`、`ip a` の出力結果、NetworkManagerのコネクション、OS上のネットワーク設定など
